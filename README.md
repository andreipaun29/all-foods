# Angular Frontend project

Setup:

    npm i
    npm i bootstrap
    npm i bootswatch

  Custom bootswatch theme used:
      https://bootswatch.com/lux/
      v 5.3

        Download bootstrap.min.css and place it inside the angular.json file
